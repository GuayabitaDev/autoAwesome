# autoBspwm2
Instalacion del entorno de trabajo (de s4vitar) automaticamente!

Probado en Debian 10, 11, Ubuntu 20.04, Parrot OS y Kali Linux

# Instalacion

```
git clone https://github.com/yorkox0/autoBspwm2
cd autoBspwm2
chmod +x autoBspwm2.sh
./autoBspwm2.sh
```
